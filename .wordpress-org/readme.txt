== Description ==

SMSI is a plugin that allows you as a shop owner to offer your customers the ability to follow their order through SMS.
With SMSI you can send a SMS to your customers on every single order status woocommerce offers even custom made ones! which grants your customers total overview of their order. You can even use variable messages which allows you to specify things like Order ID, Name, Date ordered and more!  As of January 2017 we offer SMS support from 3 different providers, we do have plans of adding more providers to list later on. Feel free to ask for providers you would like to see being added in the comments!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/smsi` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress (Note you need to have Woocommerce installed to use this plugin.)
3. Choose Smsi->settings in the sidebar of your admin panel and setup your desired provider
4. Choose Woocommerce->settings->Smsi settings and setup your messages!


== Frequently Asked Questions ==

= Do you provide support for connection issues to providers? =

No. We only provide support to our plugin. Non working API keys etc should be discussed with the provider.

= Do i need Woocommerce for this plugin? =

Yes you do this plugin extends Woocommerce's functionality.
